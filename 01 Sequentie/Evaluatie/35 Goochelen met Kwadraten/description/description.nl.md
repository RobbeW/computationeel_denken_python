## Gegeven:
Ingabire en Oskar spelen een spel. Om de beurt moeten ze een getal raden.

- Ingabire zegt een getal van **vier cijfers**.
- Oskar moet binnen 30 seconden het **nieuwe getal** bepalen dat bestaat uit de **kwadraten van elk cijfer** (in dezelfde volgorde).
- Ingabire wil dit snel kunnen controleren, dus heeft ze een programma nodig dat dit automatisch uitrekent.

## Gevraagd:
Schrijf een programma dat:

- een **getal van vier cijfers** inleest;
- de vier cijfers afzonderlijk bepaalt;
- van elk cijfer het **kwadraat** berekent;
- een nieuw getal vormt door deze kwadraten **achter elkaar te plaatsen**;
- het nieuwe getal afdrukt.

## Voorbeelden

Voor invoer:
```text
1235
```

Verschijnt er: 
```
14925
```

Voor invoer:
```text
7893
```

Verschijnt er: 
```
4964819
```
