## Gegeven
Deelbaarheid door **9** is een klassieker in hoofdrekenen.  
Een getal is deelbaar door 9 als de **som van zijn cijfers** deelbaar is door 9.  
Binnen programmeren kunnen we dat ook eenvoudig controleren met een **operator**.

Een **bijzonder geval** is `0`: dat getal is ook deelbaar door 9.

## Opgave
Schrijf een programma dat:

* Een **geheel getal** aan de gebruiker vraagt;
* Controleert of het getal **deelbaar is door 9**;
* Voor `0` ook meldt dat het **deelbaar is door 9**;
* Het resultaat op het scherm afdrukt, in dezelfde vorm als in de voorbeelden.

## Voorbeelden

Voor invoer:
```
18
```
Verschijnt er: 
```
18 is deelbaar door 9.
```

Voor invoer:
```
20
```
Verschijnt er: 
```
20 is niet deelbaar door 9.
```

Voor invoer:
```
O
```
Verschijnt er: 
```
O is deelbaar door 9.
```
