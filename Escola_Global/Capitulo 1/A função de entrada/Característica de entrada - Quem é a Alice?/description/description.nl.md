## Objetivo

Neste exercício, vai aprender a criar uma 
* Criar uma variável;
* adicionar uma string à variável;
* usar a função input;
* e imprimir o conteúdo das variáveis no ecrã.

## Atribuição

* Criar uma variável chamada `mensagem` e atribuir a string `"Olá, meu nome é"` a ela;
* Pedir ao utilizador para introduzir o seu nome. Para fazer isso, use a função `input()`;
* Adicionar o nome digitado à variável `mensagem`;
* Imprimir o conteúdo da variável `mensagem` no ecrã.

## Entrada
```
Alice
```

## Saída
```
Digite seu nome: Alice
Olá, meu nome é Alice
```
