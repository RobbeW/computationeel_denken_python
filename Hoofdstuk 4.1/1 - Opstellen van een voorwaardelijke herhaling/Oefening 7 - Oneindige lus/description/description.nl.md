## Gevraagd

Het onderstaande programma is een voorbeeld van een **oneindige lus**. Voer het uit in Dodona en onderzoek wat er fout loopt.

Wijzig deze code nu op **één plaats** zodat het onderstaande verschijnt:

```
1 : 2
2 : 6
3 : 14
```
